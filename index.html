Welcome to Simulations

Design a game economy or system as a diagram, then press Run to watch resources flow, pile up, convert, and feed back — live.

The 3-step loop

  1. Pick a node from the left palette and click the canvas to place it.
  2. Choose the Resource (R) tool, then drag from one node to another to connect them.
  3. Press Run — numbers move, tokens flow, charts draw.

New here? Browse templates to open a ready-made model and press Run.

The building blocks

Pool
stores a resource amount
Source
produces resources
Drain
consumes resources
Gate
splits flow across outputs
Converter
turns N in into a new output
Register
a value computed from a formula
Delay
holds a batch, releases it after N steps
Queue
serves items one at a time (FIFO)
Trader
swaps resources between two partners
Connections
move resources (Resource) or carry a value (State)
